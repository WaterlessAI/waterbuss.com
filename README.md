Waterbuss is an open-source initiative visualizing the invisible environmental cost of Large Language Models. Every query has a physical cost. We track it.

**Live Demo:** [https://waterbuss.com](https://waterbuss.com)

---

## ‚ú® Features

- **Real-time Consumption Engine**: Live counter estimating global AI water usage based on compute infrastructure.
- **Bento Grid UI**: A modern, responsive dashboard layout using high-fidelity glassmorphism.
- **Impact Calculator**: Interactive tool to calculate personal water footprint based on daily prompts.
- **Viral Receipt System**: Generate a "Water Receipt" to share on Twitter/X.
- **i18n Support**: Fully localized in English, Turkish, Spanish, and Chinese.
- **3D Tilt Interactions**: Physics-based card animations for an immersive experience.
- **Methodology Transparent**: Based on UC Riverside & Cornell University research (PUE 1.58, 500ml/50 queries).

## üöÄ Getting Started

To run Waterbuss locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/WaterlessAI/waterbuss.com.git

# Enter the directory
cd waterbuss.com

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
